# ks2 Newspaper

## Project setup
Set your database credentials on *./env*, like this:

    DB_HOST=hostname
    DB_USERNAME=username
    DB_PASSWORD=password
    DB_NAME=databasename
   
Run in console
```
yarn install
```
```
yarn db:migrate
```
```
yarn db:seeds
```
### Compiles and hot-reloads for development
```
yarn run dev
```
